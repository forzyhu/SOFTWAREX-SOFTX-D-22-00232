IFC algorithm

Environment collocation
1. Install libsvm. For details, see the libsvm installation guide.(./libsvm/readme)
2. After installing the libsvm, enter the code running process.

Code run
This program in the design, according to three different data sets, three data sets are ORL, YALE, MU_PIE face data set. When the three data sets are running, just go to their main method, such as Main_ORL, and click Run.

The program needs the converted data set. Make sure the data set is placed correctly.

Click "Run" and the corresponding result will appear. You can also try to change the relevant parameters in the code to view different recognition results.
